# 新築 vs 中古住宅 比較診断｜予算・条件から最適な住宅を提案 - 技術仕様書

## 概要

**サービス名**: New Vs Used Home Advisor
**バージョン**: 1.2.0
**更新日**: 2026-06-04
**URL**: https://appadaycreator.com/new-vs-used-home-advisor/

予算・条件から新築と中古の総コストを比較し最適な選択を提案。8つの質問でリフォームコスト・ローン・立地など総合的に比較する無料診断ツール。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. ページを開き、入力フォームの項目を確認する
2. 必要な情報を入力または選択する（各質問横の ℹ️ で補足説明を確認可能）
3. すべての質問に回答して結果を取得する
4. 表示された結果・分析・グラフを確認する
5. 必要に応じて「結果をコピー」ボタンで結果テキストをコピー、またはSNSで共有

## 新機能（v1.2）

### 診断精度の向上
- **Q3・Q6の診断ロジック追加**: 家族構成・ローン状況が診断に反映 → より正確な推奨結果
- **予算帯別の費用係数**: 予算により中古物件価格の割合を動的計算（2000万円帯: 60% → 5000万円以上: 72%）
- **リフォーム費用の柔軟化**: リフォーム意欲に応じて費用予測を8-20%に調整

### 結果表示の改善
- **スコア詳細の上部配置**: 診断スコアが新築vs中古で一目瞭然に
- **注意点セクションの強調**: 背景色を赤系に変更してリスク情報を明確化
- **より詳細な費用内訳**: 物件価格・リノベ費用を明確に分離表示

## 前バージョン機能（v1.1）

- **結果コピー機能**: 診断結果をテキストで簡単コピー → メール・SNSで共有
- **スムーズなリセット**: ページ全体リロードではなく、JavaScriptでスムーズに最初に戻す
- **質問補足説明**: 各質問横の ℹ️ アイコンをホバーで質問の意図を表示
- **スコア可視化**: 新築 vs 中古のスコアを横棒グラフで視覚的に表示
- **グラフ統合**: Chart.js を使用した動的スコアグラフ
